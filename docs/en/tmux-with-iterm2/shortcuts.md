# Set Shift+Enter to new line

1. Open iTerm2 > Settings > Keys > Key Bindings
2. Click the + button to add a new binding
3. Set the shortcut to Cmd+Enter (press it in the recording field)
4. Set Action to Send Text with "vim" Special Characters
5. Enter \n as the value
6. Click OK

# Set Cmd+-> to move cursor to the end of the line

1. Open iTerm2 > Settings > Keys > Key Bindings
2. Click the + button to add a new binding
3. Set the shortcut to Cmd+-> (press it in the recording field)
4. Set Action to Send Hex Code
5. Enter 0x05 as the value
6. Click OK

# Set Cmd+<- to move cursor to the start of the line

1. Open iTerm2 > Settings > Keys > Key Bindings
2. Click the + button to add a new binding
3. Set the shortcut to Cmd+<- (press it in the recording field)
4. Set Action to Send Hex Code
5. Enter 0x01 as the value
6. Click OK
